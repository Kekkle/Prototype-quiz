# Macro Munch! The Food Group Challenge

A browser-based macronutrient quiz designed for 10–12 year olds. The quiz teaches kids which foods belong to which macronutrient group (protein, carbohydrates, fats) through interactive food-icon selection.

## How to Run

Open `index.html` in any modern web browser — no installation or server required.

```
# On Linux, open from the terminal:
xdg-open index.html

# Or just double-click the file in your file manager.
```

## What's Inside

- **9 questions** mixing "pick one" and "select all that apply" formats
- Emoji-based food cards as visual answer choices
- Immediate feedback with fun nutrition facts after each question
- Score and tier badge at the end (Macro Master, Nutrition Ninja, Food Explorer, Keep Learning)
- Shuffled question order on replay

## Tech

Single HTML file with embedded CSS and JS. No dependencies, no build step.
